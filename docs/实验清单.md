# 顶刊级实验验证体系设计：物理引导的子空间 PG-ZOO

> **文档说明**：本实验设计体系旨在全面支撑“物理-拓扑双驱动”黑盒攻击框架的方法论。通过四大维度的实验，向审稿人严密论证本框架的极速逃逸能力、机制设计的必要性以及在真实工业系统中的物理合规性。

---

## 一、 核心攻防性能验证（证明“极速与高效”）

**实验目的**：回答框架在严格黑盒条件下的攻击强度与收敛速度，确立 SOTA（State-of-the-Art）地位。

### 1.1 综合性能对比分析 (Main Results Table)
* **评估指标**：攻击成功率 (ASR)、平均查询次数 (Avg. Queries)、查询次数中位数与 95 分位数 (Median / P95 Queries)。
* **对比基线 (Baselines)**：
  * **无降维原生黑盒**：如 Vanilla ZOO 或 NES（展示高维维灾难）。
  * **随机子空间黑盒**：Random Subspace ZOO（证明物理先验的价值）。
* **预期数据映射（基于现有全量测试）**：
  * **IEEE-14**：ASR 稳定在 ~98%，平均查询达 ~13.5 次。
  * **IEEE-118**：ASR 稳定在 ~90% 以上，平均查询达 ~55.5 次。

### 1.2 查询预算-成功率演化曲线 (ASR vs. Query Budget)
* **图表形式**：折线图（横轴：允许的最大黑盒查询次数 Budget；纵轴：累积攻击成功率 ASR）。
* **论证核心**：证明本框架能够在极短的查询窗口内（如 20~60 次以内）迅速收敛并完成逃逸，展示其**“单发/极少发（One-shot / Few-shot）”**的实战威慑力，有效规避 IT 层的时序速率检测（Rate Limiting）。

## 二、 核心机制的消融实验（证明“非调参、无多余设计”）

**实验目的**：剥离并验证框架中每一个物理设计（PSSM、自适应截断、轻量正则）的不可替代性。

### 2.1 PSSM 测度与自适应截断的消融
* **针对 IEEE-14 (高密度系统)**：
  * 对比：【完整 PSSM 二维局部收缩】 vs 【全空间寻优】 vs 【随机选取二维区域】。
  * 结论：盲目寻优成本暴增，随机区域 ASR 骤降；基于 PSSM 的强收缩是小系统极速逃逸的唯一解。
* **针对 IEEE-118 (稀疏大系统)**：
  * 对比：【完整 PSSM 12维物理基底】 vs 【旧版 Hard-Support 64 截断】 vs 【随机特征基底】。
  * 结论：强行使用硬截断会破坏大系统全网协同攻击骨架（查询增多且 ASR 下降），物理基底投影是大系统下兼顾效率与成功率的最优解。

### 2.2 轻量级语义正则项 ($\mathcal{P}$) 的消融
* **实验设置**：对比【包含语义保持项】与【移除语义保持项】时的 ASR 与查询次数分布。
* **预期结论**：移除该正则项后，ASR 仅产生微小波动（如 90.53% 降至 90.14%）。借此强势宣告：**语义保持不主导逃逸成功率，其被精准降级为仅用于“稳定物理骨架”的轻量正则**，避免了与核心攻击目标的梯度冲突。

## 三、 物理与隐蔽性验证（证明“工程合规性”）

**实验目的**：回应工程界与 CPS 领域审稿人最关心的物理底层逻辑，证明生成的样本不是纯计算机视觉领域的“虚假数据”，而是物理世界可运行的工况。

### 3.1 BDD 残差规避验证 (Residual Bypassing)
* **评估指标**：计算对抗样本注入前后，系统 WLS 状态估计的 $\chi^2$ 检验残差值 ($J(\hat{x})$)。
* **论证核心**：由于框架强制将扰动约束在雅可比矩阵列空间 ($a=Hc$)，残差增量始终为 0，理论且实证 100% 免疫传统 BDD。

### 3.2 SCADA 潮流越限统计 (Power Flow Security Check)
* **评估指标**：统计加入对抗扰动后，系统中各线路的实际有功潮流绝对值 ($|M(x+c_0+\Delta c)|$) 相对其热稳定极限 ($F_{max}$) 的百分比增量。
* **论证核心**：提供统计直方图，证明绝大多数由 PG-ZOO 生成的微小对抗扰动，其引发的潮流偏移完全处于电网日常运行的自然安全裕度内，没有引发大面积的硬越限报警。*(此数据用于支撑论文中的“免责声明”)*。

### 3.3 扰动能量隐蔽性 (Perturbation Stealthiness)
* **评估指标**：对抗状态扰动 $\Delta c$ 的 $L_2$ 范数，以及量测侧对抗噪声的信噪比 (SNR)。
* **论证核心**：证明攻击的高效并未以牺牲隐蔽性（注入极高能量噪声）为代价。

## 四、 跨模型泛化性实验（可选杀手锏：证明“高阶隐蔽性”）

**实验目的**：证明本框架不仅能打败基础网络，更能对冲具备空间感知能力的防御架构。

### 4.1 MLP vs. GNN 防御逃逸对比
* **实验设置**：使用同样的 PG-ZOO 框架，分别攻击基于 MLP 的非拓扑 NAD 和基于 GCN/GAT 等图神经网络的拓扑 NAD。
* **预期结论**：
  * 常规黑盒攻击生成的扰动呈“高频白噪声”，在攻击 GNN 时会被其图低通滤波机制轻易拦截，ASR 断崖式下跌。
  * 本框架生成的扰动受物理基底约束，天然呈现符合电气距离扩散的**低频平滑特性**。因此，即使面对先进的 GNN 联合防御，依然能维持 90% 以上的高逃逸率。
