# 面向DC电力系统的物理-拓扑双驱动黑盒对抗攻击框架研究

## 一、 场景描述与威胁模型

本研究聚焦于现代智能电网的能量管理系统（EMS）。在典型的状态估计（State Estimation, SE）场景中，控制中心同时部署了传统的基于物理方程的不良数据检测（BDD）和先进的基于图神经网络的异常检测器（NAD），形成了对虚假数据注入攻击（FDIA）的双重防御体系。

**威胁模型设定如下：**
1. **攻击能力与资源：** 攻击者通过入侵变电站或通信网络，获取了部分传感器节点的控制权，属于受限篡改资源下的稀疏性攻击。
2. **系统先验知识：** 攻击者完全掌握电网的直流（DC）潮流拓扑结构、线路传输容量极限、支路电纳参数以及量测雅可比矩阵$H$。
3. **黑盒攻击场景：** 攻击者**无法**获取控制中心NAD模型的内部架构、参数权重或梯度信息，仅能通过持续注入篡改数据（查询），观察控制中心的响应（是否下发调度指令或触发警报），从而获得离散的硬标签（Hard-label）或概率得分（Soft-label）反馈。

## 二、 研究动机与核心挑战

在上述严格的黑盒场景下，现有的数据驱动对抗攻击面临着不可逾越的理论与工程瓶颈，构成了本框架提出的核心动机：

1. **高维黑盒搜索的“维度灾难”与查询效率低下：** 电力系统状态空间动辄数千维，传统零阶优化（Zero-order Optimization）算法在此类高维空间中盲目搜索时收敛极慢，需要海量查询次数，在实时性要求极高的电力系统中完全不具备可操作性，且极易暴露攻击行为。
2. **纯数据扰动极易触发物理边界报警：** 现有的对抗样本生成算法（如C&W攻击、基于演化的黑盒攻击）仅通过数学范数限制扰动大小，缺乏物理语义约束的扰动叠加到真实电网数据后，极易导致线路潮流计算值突破传输极限$F_{max}$，直接触发SCADA系统的硬性越限报警，使攻击在到达NAD之前即被拦截。
3. **先进图感知防御模型的降维打击：** 目前主流的NAD普遍采用图卷积网络（GCN）等空间感知模型，若攻击者仅对孤立节点进行局部篡改，在图拓扑空间中会表现为异常的高频突变信号，GNN的邻域聚合机制可轻易识别此类异常，传统攻击的逃逸效果大幅衰减。

**动机总结：** 必须实现从“纯数据微扰”向“物理-拓扑双驱动融合”的范式转变，利用物理法则作为“硬约束”缩减搜索空间并确保工程合规，利用图谱拓扑作为“软引导”伪装攻击行为，最终实现双重防御体系的完整逃逸。

## 三、 融合框架的核心机理

本框架的作用机理分为两个相互协同、缺一不可的层级，实现了攻击可行性与隐蔽性的深度融合：

1. **底层（可行域硬约束）：** 基于DC潮流方程与线路传输极限，构建严格的线性凸多面体可行空间，将优化算法的搜索范围严格限制在该凸多面体内部，实现可行域的大幅剪枝与降维，从理论上确保生成的对抗样本天生免疫BDD残差检测与SCADA越限报警。
2. **高层（图谱域软正则化）：** 在凸多面体可行域内进行启发式寻优时，将电网拓扑图的拉普拉斯二次型作为适应度惩罚项，引导演化算法或梯度估计向符合自然物理扩散规律的低频信号区域聚集，使攻击扰动伪装成正常的负荷波动，以此欺骗具备空间感知能力的图神经网络防御模型。

## 四、 统一数学建模

在DC潮流模型下，该融合黑盒对抗攻击被建模为一个**带状态空间图正则化的受限零阶优化问题**，所有约束均为线性约束，保证了可行域的凸性与算法的收敛性。

设黑盒神经网络异常检测器为函数$\mathcal{N}(\cdot):\mathbb{R}^m\to[0,1]$，输入为$m$维量测向量$z'=z+Hc$，输出为该样本属于**正常工况**的置信度得分。攻击的核心目标是：在满足所有物理约束与资源约束的前提下，生成对抗量测$z'$，使得$\mathcal{N}(z')$最大化（即让检测器将攻击样本判定为正常，实现检测逃逸），同时通过图正则约束提升节点状态扰动的拓扑隐蔽性。

### 4.1 联合优化目标函数
$$\max_{c}\quad\mathcal{N}(z+Hc)-\lambda\cdot c^T L c\tag{1}$$

其中：
* $\lambda>0$为图正则平衡系数，用于权衡攻击逃逸性能与拓扑平滑性；
* $L\in\mathbb{R}^{n\times n}$为电网物理拓扑对应的拉普拉斯矩阵，$n$为系统节点总数；
* $c\in\mathbb{R}^n$为节点电压相角的扰动向量，是攻击的核心优化变量。

### 4.2 严格物理硬约束（搜索空间边界）
所有约束共同构成封闭的凸多面体可行域，确保攻击样本的物理合规性：

1. **量测映射约束（BDD规避）**
   $$a=Hc\tag{2}$$
   其中$a\in\mathbb{R}^m$为量测扰动向量，$H\in\mathbb{R}^{m\times n}$为DC潮流下的量测雅可比矩阵。该约束确保攻击前后状态估计的残差不变，从理论上实现对基于残差的传统BDD的100%规避。
2. **线路容量极限约束（越限报警规避）**
   $$-F_{max}\le M(x+c)\le F_{max}\tag{3}$$
   其中$x\in\mathbb{R}^n$为攻击前的基准节点电压相角，$M\in\mathbb{R}^{l\times n}$为支路潮流计算矩阵，$F_{max}\in\mathbb{R}^l$为各线路的有功潮流传输极限。该约束可标准化为线性不等式组$A_{ineq}\cdot c\le b_{ineq}$，共同切割出封闭的凸多面体可行域。
3. **攻击资源稀疏性约束**
   $$\|c\|_0\le k\tag{4}$$
   其中$k$为攻击者可入侵并篡改的最大节点数量。工程实现中，通过离线拓扑脆弱性分析预先固定$c$的非零元索引，将非凸$L_0$约束转化为固定维度的子空间搜索问题。

## 五、 黑盒攻击算法执行流水线

针对上述带约束的黑盒优化问题，本框架设计了基于受限零阶寻优（支持引入投影算子的CMA-ES或投影零阶梯度ZOO算法）的三阶段执行流水线，兼容软标签与硬标签两种黑盒反馈场景。

### 阶段 1：先验降维与凸多面体初始化（离线计算）
1. **拓扑脆弱性分析与子空间降维**
   基于电网拓扑拉普拉斯矩阵$L$，通过谱图理论识别电网弱连接区域（割集）。提取对应代数连通度的Fiedler向量，筛选数值符号翻转或绝对值接近0的节点，确定篡改向量$c$的非零元索引集合$\mathcal{S}$，将全网$n$维高维搜索极致降维至$k$维局部子群空间。这为后续零阶算法的查询效率奠定了核心基础。
2. **可行域预计算与初始种群生成**
   利用线性规划算法，预先计算由约束共同构成的凸多面体边界，通过蒙特卡洛采样在可行域内生成严格满足物理约束的初始种群。

### 阶段 2：融合图信号特征的综合评估与梯度估计（在线查询）
1. **黑盒查询反馈获取**
   * **软标签场景**：算法将生成的候选样本提交至检测器$\mathcal{N}(\cdot)$，获取概率得分，并通过有限差分（如ZOO算法）估计局部梯度。
   * **硬标签场景**：通过边界游走策略将离散标签转化为排序得分，适配演化算法需求。
2. **图谱联合惩罚与半白盒解析**
   在本地计算当前候选样本的状态图平滑度$c^T L c$。在梯度优化时，该图正则项的梯度可直接解析求出，无需黑盒查询，从而形成“黑盒估计+白盒解析”的极高寻优效率。

### 阶段 3：受限子空间内的零阶更新与边界投影
1. **扰动变量更新**
   根据阶段2获取的反馈，利用演化策略（如CMA-ES）或动量梯度下降（如Adam）生成下一代候选扰动向量$c_{temp}$。
2. **凸多面体边界投影**
   对$c_{temp}$执行物理合规性检验，若越界跳出凸多面体，则触发投影算子，将其强制拉回至多面体边界上最近的可行点：
   $$\min_{c}\quad\|c-c_{temp}\|_2^2\tag{5}$$
   $$\text{s.t.}\quad Hc=a,\quad A_{ineq}c\le b_{ineq}$$
   该投影为带线性约束的二次规划（QP）问题，存在唯一全局最优解，确保迭代全程零违规查询。

## 六、 核心创新点凝练
1. **范式创新**：首次提出物理-拓扑双驱动的黑盒攻击新范式，实现了「BDD规避+越限免疫+GNN逃逸」的三重攻击目标。
2. **半白盒优化理论**：将目标函数拆分为黑盒检测项与白盒拓扑正则项，证明了在引入P-ZOO算法时，可利用精确解析的图梯度引导零阶估计方向，极大地提升了软标签场景下的寻优效率。
3. **多维降维算法设计**：通过谱图理论离线降维完美化解了ZOO算法与演化算法在高维系统中的查询次数灾难，并通过边界投影算子保证全流程物理合规性。
4. **递进式工程落地**：框架设计兼容从基础MLP到高阶GNN的跨模型攻击，同时支持硬标签与软标签场景，具备极强的学术严谨性与真实EMS适配性。

## 七、 完整实验方案设计

### 7.1 实验环境与测试系统
* **仿真平台**：MATLAB（MATPOWER用于电网仿真与物理投影）+ Python（PyTorch/PyG用于NAD构建与对接）。
* **标准测试系统**：IEEE 14/30节点（算法快速验证）、IEEE 118节点、波兰2383节点（验证维度灾难克服能力）。

### 7.2 目标防御体系构建（递进式架构）
1. **前置BDD模块**：WLS状态估计与$\chi^2$不良数据检测（95%置信度）。
2. **阶段一：MLP-Based NAD（基础防御基线）**
   构建基于多层感知机的非拓扑感知检测器。主要用于快速跑通跨平台数据交互接口，并验证物理边界投影算子在底层收敛性上的有效性，确保攻击样本100%满足SCADA越限约束。
3. **阶段二：GNN-Based NAD（进阶空间感知防御）**
   在阶段一验证通过后，将目标升级为GCN、GAT与GraphSAGE三种主流图异常检测器，验证拓扑正则项的逃逸效果。

### 7.3 对比基准算法
* **基准1：经典FDIA**：传统物理驱动攻击，仅实现BDD规避。
* **基准2：C&W白盒攻击**：基于梯度的理论上限基准。
* **基准3：原生黑盒攻击**：纯数据驱动零阶优化攻击（原生ZOO或NES）。
* **基准4：受限黑盒攻击（Constrained Baselines）**：在原生算法中强制加入本框架的物理投影算子，但不包含图正则项，用于剥离证明在对抗GNN时，单靠数值约束无法隐蔽高频拓扑噪声。

### 7.4 核心评价指标
* **攻击性能**：攻击成功率（ASR）、BDD规避率（理论值100%）、越限报警率（理论值0%）。
* **攻击隐蔽性**：量测扰动$L_2$范数、图平滑度得分$c^T L c$。
* **攻击效率**：平均黑盒查询次数（衡量实时性与暴露风险）。

## 八、 工程实现关键细节

### 8.1 凸多面体投影算子实现
* 中小规模系统：采用MATLAB `quadprog` 或 Python `cvxopt` 直接求解QP问题。
* 大规模系统：采用交替方向乘子法（ADMM）求解投影问题，避免高维矩阵求逆开销。

### 8.2 黑盒核心求解器适配（含ZOO算法扩展）
本框架在阶段三的扰动更新中，支持针对不同反馈标签场景的求解器平滑切换：
1. **P-ZOO（投影零阶优化算法）—— 软标签场景的效率利器**
   * **半白盒解析优势**：目标函数$J(c)=\mathcal{N}(z+Hc)-\lambda c^T L c$的梯度可表示为$\nabla_c J \approx \hat{\nabla}_c\mathcal{N}-2\lambda L c$。其中图正则部分的梯度$2\lambda L c$是精确的白盒解析梯度。
   * **降维契合**：ZOO算法原生的维度灾难被阶段一的Fiedler降维彻底化解，只需对保留的$k$个维度（如10维）进行对称有限差分估计，单次迭代仅需$2k$次查询。利用估计出的混合梯度与Adam动量更新后，再执行QP投影，使得收敛速度远超纯启发式算法。
2. **带投影的CMA-ES / PSO —— 硬标签/离散反馈的通用底座**
   * 在防御方仅返回0/1警告时，ZOO算法由于无法差分估计梯度而失效。此时切换至带投影的CMA-ES或PSO，将综合适应度直接作为群体优化的目标，每一代种群更新后统一执行物理越界点拉回。

### 8.3 离线拓扑脆弱性分析与降维
1. 构建加权拉普拉斯矩阵$L=D-A$（权重为支路电纳）。
2. 获取第二小特征值对应的Fiedler向量。
3. 提取数值符号翻转或绝对值近乎为0的节点（割集瓶颈），将其作为$c$的非零元索引，固定这$k$个维度，使全网高维搜索被严格限缩。

## 九、 关键理论深化

### 9.1 可行域凸性与状态图正则的物理意义
* **凸性证明**：等式残差约束、线路越限不等式约束以及固定非零元索引的稀疏性约束，其对应的可行集均为凸集，有限个凸集的交仍为严格凸多面体，保证了投影算法的唯一全局最优。
* **物理意义**：图拉普拉斯二次型$c^T L c = \frac{1}{2}\sum_{(i,j)\in\mathcal{E}}B_{ij}(c_i-c_j)^2$直接衡量相邻节点相角扰动的差异。该值越小，代表相角篡改量在拓扑空间中越平滑，完美符合“电气距离越近，相角差越小且无突变”的物理定律，从而精准对冲GNN的图滤波与邻域聚合机制，比直接约束量测分布具备更强的可解释性与更低的矩阵运算开销。

## 十、 论文摘要示例
现代智能电网能量管理系统（EMS）普遍部署了基于物理方程的不良数据检测（BDD）与基于图神经网络的异常检测器（NAD），形成了对虚假数据注入攻击（FDIA）的双重防御体系。现有黑盒对抗攻击方法面临高维搜索维度灾难、易触发物理越限报警、无法逃逸图感知防御的核心瓶颈。针对上述问题，本文提出了一种物理-拓扑双驱动的黑盒对抗攻击新框架。首先，基于直流（DC）潮流方程与线路传输极限，构建了严格的凸多面体可行域，将优化搜索空间严格限制在可行域内，保证了生成的攻击样本100%规避BDD与SCADA潮流越限报警；其次，引入电网物理拓扑的拉普拉斯二次型作为软正则项，约束节点相角篡改量在空间上的平滑度，引导攻击向量伪装成符合自然潮流扩散规律的低频波动信号；最后，设计了兼顾P-ZOO（投影零阶优化）与演化算法的受限流水线，结合谱图理论离线降维、半白盒梯度解析与凸多面体边界投影，大幅解耦了高维寻优难度。基于多规模IEEE标准测试系统的跨模型递进式实验表明，所提框架不仅在基础MLP防御下收敛极快，且在面对进阶的GNN防御时，其攻击成功率显著优于现有受限与非受限黑盒算法，实现了对双重防御体系的完美逃逸，具备极强的理论支撑与工程泛化性。
