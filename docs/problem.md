# 论文方法论：基于物理降维与子空间PG-ZOO的单发黑盒NAD逃逸

## 1. 研究问题与攻击模型

考虑直流（DC）状态估计下的黑盒神经网络异常检测器（NAD）攻击。系统的测量模型记为：

$$z = H\theta + e$$

其中，$z \in \mathbb{R}^{m}$ 为测量向量；$\theta \in \mathbb{R}^{n}$ 为状态向量（节点电压相角）；$H \in \mathbb{R}^{m\times n}$ 为 DC 雅可比矩阵；$e$ 为测量噪声。

为了保证对抗攻击在物理上的隐蔽性（即规避传统的基于残差的不良数据检测 BDD），本文并非直接在测量空间无约束地构造扰动，而是在已有虚假数据注入攻击（FDIA）状态 $c_0$ 的基础上叠加对抗修正 $\Delta c$：

$$c = c_0 + \Delta c$$

对应的测量侧注入向量为：

$$a = Hc = H(c_0+\Delta c) = a_0 + H\Delta c, \quad a_0 = Hc_0$$

因此，送入目标 NAD 的最终对抗样本为：

$$z_{\mathrm{adv}} = z + a_0 + H\Delta c$$

该构造方式确保了新增的对抗扰动始终位于雅可比矩阵的列空间 $\mathrm{col}(H)$ 内，从理论上保持了 FDIA 完美的物理可实现结构。

## 2. 现有黑盒攻击的理论瓶颈与工程局限

尽管现有的零阶优化与降维攻击在特定系统上取得了一定的成功率，但若要将其应用于高安全等级的电力信息物理系统（CPS），当前的研究范式与理论框架仍存在以下四个核心瓶颈（即本文致力于解决的问题）：

1. **降维逻辑的理论割裂（缺乏泛化性）**：现有的黑盒降维攻击通常因系统规模而异。例如，在小系统中往往采用简单的硬截断或区域筛选，而在大系统中又不得不采用复杂的全局降维或稀疏支持集。这种因测试用例而异的底层逻辑，属于典型的“过拟合调参”，无法形成一套具有数学普适性的统一降维理论。
2. **物理先验缺乏严格的数学支撑**：为了加速黑盒查询，大量研究尝试引入物理先验。然而，现有方法多依赖于将拓扑度数、雅可比范数等指标进行启发式（Heuristic）的线性加权或连乘，并引入人为指定的超参数。这种缺乏严谨物理可解释性的“公式拼凑”，严重削弱了攻击算法的理论深度。
3. **致命的物理合规性漏洞（忽视 SCADA 越限）**：绝大多数针对 NAD 的对抗攻击仅关注如何保证残差不变（规避 BDD）和欺骗分类器，却完全忽略了电力系统的绝对物理安全极限。如果叠加对抗扰动 $\Delta c$ 后，导致某条关键线路的热极限越限（即 $|M(x+c_0+\Delta c)| > F_{max}$），SCADA 系统会直接触发硬性报警甚至继电保护跳闸。这种引发物理警报的攻击即使骗过了 NAD 也是失效的。
4. **攻击目标的主次混淆（正则化定位不清）**：在复杂攻击的优化目标设计中，现有工作往往将“维持原始 FDIA 语义一致性”与“逃避 NAD 检测”视为同等重要的核心目标，这会导致零阶估计的梯度方向产生严重冲突，极大增加了黑盒查询的迭代成本。

*(注：针对上述第三点物理越限漏洞，本文重点聚焦于提出对抗双重防御体系的统一理论框架。对于严格线路热极限的凸多面体投影机制，我们在理论模型中进行了预留，但在本轮实验中假设微小扰动处于自然裕度内，带投影的优化求解将作为未来的扩展工作。)*

## 3. 方法总览：物理引导的单发子空间 PG-ZOO

针对上述瓶颈，本文提出一种名为 **Physics-guided one-shot subspace PG-ZOO attack** 的高效黑盒攻击范式。其核心流水线由三步组成：

1. **构建统一物理测度**：摒弃启发式拼凑，利用具有严谨、恒定物理意义的联合测度对状态维度进行评估，一次性（One-shot）确定最终的极致降维搜索子空间；
2. **搜索域严格限制**：后续所有的黑盒查询与演化更新，均严格限制在该极低维物理子空间内完成（不采用“失败后扩维重来”的试错机制，以最小化查询开销）；
3. **子空间零阶寻优与分级约束**：在子空间中运用投影零阶梯度优化（PG-ZOO）估计局部下降方向，最小化 NAD 决策边界 margin，并明确将 FDIA 语义保持项降级为辅助的“轻量级正则化”。

## 4. 物理子空间重要性测度（PSSM）构建

为了在不进行任何黑盒查询的前提下精准定位最脆弱的攻击维度，本文提出**物理子空间重要性测度（Physical Subspace Significance Measure, PSSM）**。

有别于依赖传感器噪声统计特性的方法，本测度严格锚定于电网恒定不变的底层物理信息（即由网络拓扑与支路电纳推导出的雅可比矩阵 $H$）。记 $h_i$ 为矩阵 $H$ 的第 $i$ 列，代表第 $i$ 个节点状态对全网测量空间的物理响应模式。定义状态在测量流形上的纯物理增益为：

$$g_i = \|h_i\|_2$$

我们将联合测度分数 $s_i$ 严谨地拆解为三个具有明确物理意义的因子连乘：

$$s_i = \operatorname{Norm}\left( \mathcal{A}_i \cdot \mathcal{M}_i \cdot \mathcal{T}_i \right)$$

1. **攻击锚点因子 (Attack Anchor Factor) $\mathcal{A}_i = |(c_0)_i|$**：
   对抗扰动 $\Delta c$ 不能脱离原始攻击的物理骨架。该因子确保零阶优化的搜索能量优先集中在原始 FDIA 已经激活的状态上，天然满足篡改稀疏性。
2. **流形敏感度因子 (Manifold Sensitivity Factor) $\mathcal{M}_i = g_i$**：
   直接反映了状态变量 $i$ 对全局测量流形（Measurement Manifold）的纯物理驱动力。该值越大，说明在没有任何传感器权重干扰下，仅需极小的状态篡改便能在测量空间引发剧烈的状态偏移，从而跨越 NAD 的分类边界，极大提升单步查询收益。
3. **拓扑中心性正则 (Topological Centrality Regularization) $\mathcal{T}_i = 1+\alpha \frac{d_i}{d_{\max}}$**：
   其中 $d_i$ 为不变的节点物理连通度，$\alpha=0.1$ 为一阶微扰系数。在低维搜索中，高连通度节点（Hubs）能将扰动更高效地传导至广域物理空间中，该正则项作为一阶泰勒展开形式的微扰，用于在敏感度相似的节点间打破平局。

## 5. 基于可观测性密度的自适应子空间截断

针对不同规模的电力系统在拓扑稀疏性与局部可观测密度上存在的本质差异，本文基于 PSSM 提出统一的**自适应子空间截断策略**，彻底解决理论割裂问题。

### 5.1 高密度耦合系统（如 IEEE-14）：极度局部收缩
在小规模系统中，雅可比矩阵 $H$ 高度稠密，节点间电气耦合极强。一个极小维度的紧凑区域所张成的测量子空间，已具备足够大的基数（Rank）来跨越 NAD 决策边界。因此，采用极强收缩：
* **截断策略**：选取测度最高的连续二维局部区域 $\Omega^\star$。
* **子空间定义**：$\mathcal{S} = \Omega^\star$。
* **物理本质**：通过物理先验锁定最有效的局部入口，彻底剔除无效的零阶探测方向，实现查询次数的极致压缩。

### 5.2 稀疏局部可观测系统（如 IEEE-118）：物理基底映射
在大规模系统中，网络呈现典型的局部稀疏特性。单一的局部微小区域缺乏撼动全局 NAD 边界的动能。此时，将物理先验用于构造低维物理特征基底：
* **截断策略**：保留更宽泛的特征支撑集（如 Top-64 Support），随后在其上提取由物理先验诱导的 12 维物理基底矩阵 $U \in \mathbb{R}^{12\times n}$。
* **子空间定义**：优化变量 $\Delta c$ 被映射为 $\Delta c = U^\top \alpha, \quad \alpha\in\mathbb{R}^{12}$。
* **物理本质**：并非割裂方法，而是由大系统固有的稀疏性决定的自适应演化。不破坏全网协调攻击骨架，将高维无序的随机搜索安全投影到低维的高敏物理基底上。

## 6. 分级优化的黑盒目标与轻量级语义正则

设 NAD 对攻击类和正常类的 logit 分别为 $f_1(z)$ 与 $f_0(z)$，定义分类 margin：

$$M(z)=f_1(z)-f_0(z)$$

在确定的子空间 $\mathcal{S}$ 内，本文明确区分了主驱动力与辅助约束，将黑盒攻击转化为无梯度的分级联合优化问题：

$$\min_{\Delta c \in \mathcal{S}} M(z+a_0+H\Delta c)+\lambda \mathcal{P}(\Delta c;c_0)$$

其中 $\mathcal{P}(\cdot)$ 为 **FDIA 语义保持项**。本文明确将其降级为一种**轻量级正则化（Lightweight Regularization）**，它不主导 ASR 的提升，仅旨在约束 $\Delta c$ 为“围绕原始骨架的微小修正”，防止对抗噪声破坏原始攻击者的初衷。具体包括三个软约束分量：
1. **状态方向保持**：$\rho_s= \frac{\langle c^{(\mathrm{supp})}, c_0^{(\mathrm{supp})}\rangle}{\|c_0^{(\mathrm{supp})}\|_2^2}$
2. **测量方向保持**：$\rho_z= \frac{\langle a, a_0\rangle}{\|a_0\|_2^2}$
3. **支撑外能量抑制**：$\rho_{\mathrm{off}}= \frac{\|c^{(\mathrm{off})}\|_2}{\|c_0^{(\mathrm{supp})}\|_2}$

总惩罚项 $\mathcal{P}$ 通过 ReLU 型函数 $[\cdot]_+$ 对违反下/上限的偏离进行二次惩罚。

## 7. 子空间 PG-ZOO 的零阶求解

在最终子空间 $\mathcal{S}$ 中，本文采用基于协方差与预条件器增强的 PG-ZOO 算法：

1. **两点查询估计**：设当前扰动为 $\Delta c^{(t)}$，在基底方向 $u_j$ 上以探测步长 $\mu_t$ 进行对称差分查询，估计子空间方向导数 $\hat{g}_j$：
   $$\hat{g}_j = \frac{F(\Delta c^{(t)}+\mu_t u_j)-F(\Delta c^{(t)}-\mu_t u_j)}{2\mu_t}$$
2. **结构化方向增强**：为避免零阶估计引入高频噪声，引入结构化协方差（指导采样方向分布）与物理预条件器（近似局部 Fisher 信息调整更新步长）。
3. **线搜索更新**：组合多个方向导数后得到最优下降方向，在子空间内执行线搜索更新。
