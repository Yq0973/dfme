# 优化空间与约束空间错位问题审查与重构指南

## 1. 核心理论漏洞诊断：空间错位

审查当前代码库（特别是在 `attack.py` 中关于扰动约束的实现）发现，PG-ZOO 算法存在严重的**优化空间与约束空间错位（Spatial Mismatch）**问题。

### 1.1 漏洞现象
- **探索与投影发生在状态空间**：在 ZOO 的每一步更新后，调用了 `_project_l2(delta_c, radius)`，算法认为自己在一个高维标准球体（Sphere，$\|\Delta c\|_2 \le R$）内进行安全探索。
- **最终隐蔽性硬约束在测量空间**：通过 `_apply_measurement_l2_cap`，要求最终测量扰动满足 $\|H \Delta c\|_2 \le C$。

### 1.2 导致的数学灾难
1. **几何流形冲突（球体 vs. 椭球）**：
   在状态空间中，测量约束 $\|H \Delta c\|_2 \le C$ 等价于 $\Delta c^\top (H^\top H) \Delta c \le C^2$。这是一个严重倾斜且扁平的椭球（Ellipsoid）。PG-ZOO 在一个球体内盲目采样，极易频繁跳出实际安全的椭球边界，导致海量黑盒查询被浪费在绝对不可行的区域。
2. **零阶梯度估计被“暴力缩放”污染**：
   如果在目标函数内部调用 `_apply_measurement_l2_cap`，超出边界的探测点会被强行按比例缩放回边界。这破坏了 ZOO 算法对称有限差分 $\frac{f(c + \mu u) - f(c - \mu u)}{2\mu}$ 中的固定步长 $\mu$，导致估计出的梯度方向彻底异化为噪声。
3. **事后缩放摧毁隐蔽性（非线性坍塌）**：
   如果仅在优化结束后执行 `_apply_measurement_l2_cap`，将跨越决策边界的对抗样本强行按比例缩小 $\alpha z_{adv}$（$\alpha < 1$），由于深度神经网络（MLP/GNN）的决策边界是高度非线性的，这一缩放极易将样本拉回“正常区域”，导致攻击在最后一步物理校验时失败。

## 2. 数学重构方案：测量隐空间正交基底变换（SVD）

为了彻底消除上述漏洞，必须**将优化变量的约束流形转化为标准的欧几里得球体**。当前学术界最严谨且高效的 SOTA 方案是利用奇异值分解（SVD）进行正交基底变换。

### 2.1 数学推导
设降维后的物理基底为 $U_K$（即代码中的 `state_basis_vectors`），优化系数为 $\alpha$。
映射关系为：$\Delta c = U_K \alpha$。
注入测量空间的扰动为：$\Delta z = H U_K \alpha$。

令投影矩阵 $H_{sub} = H U_K$。隐蔽性约束为：
$$\|H_{sub} \alpha\|_2 \le C$$

**步骤 1：SVD 分解**
对 $H_{sub}$ 进行奇异值分解：
$$H_{sub} = U_{svd} \Sigma V_{svd}^\top$$
其中，$U_{svd}$ 的列向量在测量空间中相互正交。

**步骤 2：定义隐空间优化变量 $\beta$**
我们放弃直接优化 $\alpha$，转而定义新的优化变量 $\beta$：
$$\beta = \Sigma V_{svd}^\top \alpha$$
此时，测量空间的扰动可完美表达为：
$$\Delta z = U_{svd} \beta$$

**步骤 3：球体约束的等价转换**
由于 $U_{svd}$ 是列正交矩阵，测量空间的 L2 范数完美等价于 $\beta$ 空间的 L2 范数：
$$\|\Delta z\|_2 = \|U_{svd} \beta\|_2 = \|\beta\|_2$$
**结论**：测量空间的隐蔽性硬约束 $\|H \Delta c\|_2 \le C$，在 $\beta$ 坐标系下，被完美转化为一个**标准的、等距的高维球体约束 $\|\beta\|_2 \le C$**。
